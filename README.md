## Project Overview

This repository contains the group project for MATH1604. The goal is to analyse a dataset of 64 respondents' answers to a 100-question multiple-choice Python quiz, and investigate whether the quiz setter arranged the correct answers in a deliberate, non-random pattern.

The project is divided into four modules, each owned by a team member, and integrated into a single analysis pipeline.

## How to Run the Full Pipeline

### 1. Clone the repository

```bash
git clone https://github.com/pshq0776-uol/MATH1604-group-project.git
cd MATH1604-group-project
```

### 2. Install dependencies

All modules use the Python standard library except for `data_analysis_M3.py`, which requires `matplotlib` and `scipy`:

```bash
pip install matplotlib scipy
```

### 3. Run the pipeline

```bash
cd scripts
python run_full_analysis_M4.py
```

This will:
1. Create `data/` and `output/` directories if they don't exist
2. Download all 64 respondent answer files from GitHub
3. Collate them into `output/collated_answers.txt`
4. Extract and save individual answer sequences
5. Compute mean answer values per question
6. Display scatter and line plot visualisations

## Module Descriptions

### `data_extraction_M1.py` — Team Member 1

Provides two functions:

- **`extract_answers_sequence(file_path)`**  
  Parses a raw quiz answer `.txt` file and returns a list of 100 integers (1–4 for selected answers, 0 for unanswered questions).

- **`write_answers_sequence(answers, n, destination_path)`**  
  Writes an extracted sequence to `answers_list_respondent_n.txt` in the specified folder.

---

### `data_preparation_M2.py` — Team Member 2

Provides two functions:

- **`download_answer_files(cloud_url, path_to_data_folder, total_respondents)`**  
  Downloads answer files from the cloud repository and saves them locally. Handles missing files (HTTP 404) gracefully.

- **`collate_answer_files(data_folder_path)`**  
  Merges all individual respondent files into a single `collated_answers.txt`, with sections separated by `*`.

---

### `data_analysis_M3.py` — Team Member 3

Provides two functions:

- **`generate_means_sequence(collated_answers_path)`**  
  Returns a list of 100 floats — the mean answer value per question across all respondents (excluding unanswered questions).

- **`visualize_data(collated_answers_path, n)`**  
  Produces a scatter plot of means (n=1) or a line plot of all individual answer sequences (n=2).

---

### `run_full_analysis_M4.py` — Team Member 4 (Team Leader)

Integrates all three modules into a complete, executable pipeline. Run this script to perform the full analysis from download to visualisation.
